# azure-ad-m365-infrastructure-lab
Azure Active Directory & Microsoft 365 Infrastructure Lab
Project Overview

This project demonstrates the deployment of a simulated enterprise IT infrastructure using Microsoft Azure, Windows Server, Active Directory, and Microsoft 365.

The environment was designed for a college infrastructure scenario and includes identity management, access control, Group Policy configuration, and Microsoft 365 administration.
